Converts a wrapper class to its corresponding primitive class.

This method checks if the given Class object represents a wrapper type (e.g., Integer, Long, Double, etc.) and returns the corresponding primitive type (e.g., int.class, long.class, double.class, etc.). If the provided type is not a wrapper class, it is returned unchanged.
